Feels unlikely that there is anyone using this at all other than me or me on my laptop but this is my macro for grinding monkey knowledge in btd6

In order to use it you have python installed and run "pip install -r requirements.txt"
Then simply run main.py (with the command "python main.py") in a terminal

Before running the macro, you must ensure that you have calibrated the positions as unless your monitor is EXACTLY the same size as mine it is very unlikely to work as intended.

While in use there are options to change settings like delays between inputs to accomodate for lag and you can make sure it has the right keybinds to work with your game.

To make it stop running: use ctrl+C in ther terminal or move your mouse in the corner of your screen for like 2-5 seconds to trigger the pyautogui failsafe

If you find any problems or have suggestions for features you can use a github issue to lmk but it feels unlikely you will.
I plan on making a tkinter UI for it so that it is slightly easier to use.
